Project Overview

This project involved developing a comprehensive monitoring and control system for Coca-Cola's production plant. The solution enables precise control of the entire manufacturing process according to specific product and recipe requirements, ensuring consistent quality and operational efficiency.

The system delivers real-time monitoring capabilities with advanced statistical analysis, including automatic calculation of Critical Process (CP), Process Capability Index (CPK), and similar quality metrics. This data-driven approach enables production managers to maintain optimal production parameters and quickly identify potential issues before they impact product quality.

A key feature of the system is its robust reporting functionality, which generates both automatic and manual reports according to shift requirements. These reports provide valuable insights into production performance, quality metrics, resource utilization, and operational efficiency.

The project encompassed complete SCADA development, intuitive UI design, comprehensive reporting systems, and secure data storage in MySQL databases. This end-to-end solution delivers a powerful tool for production management while maintaining the high quality standards essential to the Coca-Cola brand.

Key Features

Process Control

Comprehensive control of the entire production process according to specific product and recipe requirements, ensuring consistent quality and efficient operation.

Statistical Quality Control

Automated calculation of Critical Process (CP), Process Capability Index (CPK), and similar statistical metrics to maintain optimal production parameters and quality control.

Shift Reporting

Flexible reporting system that generates both automatic and manual reports according to shift requirements, providing comprehensive insights into production performance.

Alarm Management

Real-time display of live and critical alarms with priority-based notifications to ensure prompt response to production issues and minimize downtime.

SCADA Implementation

Complete development of Supervisory Control and Data Acquisition (SCADA) system with intuitive user interface design tailored to production floor requirements.

Data Management

Robust data storage solution using MySQL databases with optimized schema design for efficient data retrieval, analysis, and long-term storage of production metrics.

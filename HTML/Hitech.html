Project Overview

I developed a robust substation monitoring and energy management system for Hitech Elex LLP, leveraging the capabilities of SpandanSCADA and the MODBUS 485 communication protocol. This system is designed for seamless acquisition, visualization, and reporting of electrical data from a network of advanced multifunction meters (MFM) and ABT meters.

The monitoring solution focuses on real-time acquisition from 30+ energy meters, each capturing 13 distinct parameters, ensuring accurate and reliable data flow throughout the facility. The communication backbone of the project is built on MODBUS 485, providing efficient and secure connectivity across all meters.

The solution includes the development and configuration of 43 custom-designed SCADA screens, purpose-built for data acquisition, visualization, energy monitoring, and report generation. These interfaces empower operators with clear insights into system health and performance.

Key components of the project include:

  • Integration of 33 meters, communicating over MODBUS 485
  • Acquisition and visualization of key electrical parameters
  • Data logging and storage in a lightweight SQLite database
  • Tailored report generation based on shift timings
  • Automated email dispatch of daily reports
  • Real-time SMS alerts for critical alarms and fault conditions

The project involved complete development of the SCADA system with intuitive UI design, comprehensive reporting capabilities, and reliable data storage utilizing SQLite database architecture. One of the key features is the automatic report generation according to shift schedules, complemented by automated email distribution and SMS alerts for critical alarms.

The user-friendly interface was meticulously designed to provide operators with immediate visibility into substation conditions, enabling efficient monitoring and quick response to any anomalies or emergency situations.

Key Features

Critical alarms instantly communicated through SMS and email notifications to ensure rapid response regardless of operator location.

SQLite database implementation for efficient data storage, enabling historical analysis, trend identification, and predictive maintenance capabilities.

Comprehensive event recorder captures all system activities, operator actions, and equipment state changes for compliance and troubleshooting purposes.

Data Management

Reporting System
